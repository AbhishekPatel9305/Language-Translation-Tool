# 🌐 CodeAlpha - Language Translation Tool

This is a submission for the **Artificial Intelligence Internship at CodeAlpha**.

## 🔍 Overview
A simple and interactive **language translation web app** built using Python and Streamlit. Users can:
- Enter text in any language.
- Choose source and target languages.
- View the translated output.
- Listen to the translated speech.

## 🛠️ Tech Stack
- Python
- Streamlit (for UI)
- Googletrans (translation)
- gTTS (text-to-speech)
